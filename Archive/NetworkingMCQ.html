direct routing:
if two computers on the same network need to communicate, the packets dont require a router. in an IEEE 802.3 network, the sending entity encapsulates the packet in an Ethernet frame, binds the destination address, and transmits the resulting frame directly to its destination. this is called direct routing which is performed by ARP
indirect routing:
If two computers are not on the same network, they must send the IP packet to the router for delivery. this is called indirect routing
static routing vs dynamic routing
a static router contains routing information tables that must be built and updated manually. If a certiain route doesnt exist in the table, it will be unable to communicate with that network

a dynamic router on the other hand communicates with other dynmic routers nto calculate routes automatically using routing protocols such as RIP and Open shortest path first (OSPF)
